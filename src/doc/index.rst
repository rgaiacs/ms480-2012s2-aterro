Bem-vindo a Experimentos Computacionais de MS480 - 2012S2 - Aterro com Obstáculo!
=================================================================================

Essa página fornece informações sobre a implementação computacional para o
Problema do Aterro com Obstáculo disponível em
https://github.com/r-gaia-cs/ms480-2012s2-aterro.

Se você está procurando saber o que é o Problema do Aterro em
http://www.youtube.com/watch?v=KPea6ukb1Y8&feature=player_embedded você encontra
um vídeo do Prof. Martinez explicando esse problema.

Se está procurando informações teóricas sobre o Problema do Aterro com
Obstáculo, você encontrará isso no :download:`relatório
<../../relatorio/relatorio.pdf>`.

Se estiver procurando os slides utilizado na apresentação, você encontrará isso
na :download:`apresentação <../../apresentacao/apresentacao.pdf>`.
